Пожалуйста, НЕ ИЗМЕНЯЙТЕ пароль непосредственно
в config.php, используя текстовый редактор, поскольку
config.php содержит пароль в зашифрованном виде.
Изменяйте пароль правильным способом, войдя ОНЛАЙН
в панель управления (панель администратора сайта)!!!
(Войти в панель можно, указав пароль по умолчанию "test".)

Это наиболее легкий и лучший способ.

ВАЖНО! Для редактирования файлов используйте
ИСКЛЮЧИТЕЛЬНО текстовый редактор, поддерживающий
обнаружение, открытие и сохранение файлов
в кодировке "utf-8 without BOM" (Byte Order Mark),
например, notepad++.

Если это условие не будет соблюдено, вы можете
повредить установку CMSimple_XH.

Если Вы забыли свой пароль, Вы можете использовать
ftp и ввести в config.php зашифрованный пароль "test",
например:

\$P\$BHYRVbjeM5YAvnwX2AkXnyqjLhQAod1

После этого Вы можете войти в панель управления,
используя пароль "test".

CMSimple_XH - это быстрая, компактная, легкая
в установке и использовании модульная Система
Управления Содержимым (Content Management
System, CMS), не требующая базы данных.

CMSimple_XH распростаняется на условиях свободного
программного обеспечения по лицензии GPL3.

CMSimple_XH содержит страницы сайта в отдельном
файле html (content.htm).

По умолчанию, теги заголовков H1-H3 используются
для разбиения содержимого файла content.htm
на отдельные страницы и создания навигационного
меню, а теги заголовков H1-H3 предназначены для
для обычного семантического структурирования
содержимого в пределах страниц.
